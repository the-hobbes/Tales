This site is about telling stories.

A final project for CS148 (database design for the web), this site allows users to sign up and log in to view stories created by other users.

I've currently built the registration, logon, user and storyteller accounts, password reset, and story creation parts of the page. What's left are editing the stories, and displaying them.

The greatest difficulty I've had with this project was an error in the approach: I came at it like a website, when I should have viewed it as a program instead.

Although I wish I had made that realization earilier, I've attempted to correct my mistakes. I've shifted to a much more object-oriented handling of processes in the code now, and the site is certainly better for it.

The project isn't quite done yet. Once the functionality is done, I can proceed with beautification. That includes things like a parallax style storytelling framework and expansion of storyteller modules. I'll leave you with a few sites that provided inspiration for the project.
